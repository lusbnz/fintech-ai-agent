## Overview

**Finn** is a cross-platform mobile application designed to help users take control of their personal finances with ease and intelligence.

Key goals:

- Quickly record & categorize transactions  
- Visually track budgets by category  
- Receive smart alerts for overspending & personalized saving tips  
- Chat naturally with an **AI financial assistant** to create transactions, view reports, get advice  
- Export PDF statements & attach receipt photos  

Built with native iOS (SwiftUI) + native Android (Jetpack Compose), unified backend, and integrated **Large Language Models** for a truly smart, personalized experience.

## Highlighted Features

| Category          | Main Features                                                                    | Notes                              |
|-------------------|----------------------------------------------------------------------------------|------------------------------------|
| **Core**          | Manual transaction entry • Budget management • Custom categories                 | Available on both iOS & Android   |
| **AI-Powered**    | AI chat assistant (create transactions, analyze spending, give suggestions) • Voice input | Powered by LLM APIs               |
| **Automation**    | Recurring transactions • Auto top-up budgets • Overspending warnings             |                                    |
| **Reporting**     | Charts by time / budget / category • PDF export with AI insights                 | AI-generated analysis included    |
| **Extras**        | Multi-device sync (FCM push notifications) • Feedback submission • Receipt image storage |                                    |
